# Friendly Parakeet Design Assets

## Menu Bar Interface
Design for the macOS menu bar dropdown, showing:
- Icon states (Idle, Active, Warning, Alert)
- Dropdown menu structure
- Notification types and sounds

## Parakeet Icon Designs

### V1 - Geometric Friendly
Initial design exploration with geometric shapes.

### V2 - Budgie Style (Recommended)
Authentic budgie design featuring:
- Round spherical head
- Yellow face with dark crown stripes
- Purple/blue cheek spots
- Blue cere (nostril area)
- Short curved beak
- Green body with wing barring
- Long blue tail feathers

## Color Mutations (Themes)
Based on real budgie color genetics:
- Wild Green (Default)
- Sky Blue (Active)
- Lutino/Yellow (Warning)
- Violet (User choice)
- Cobalt (User choice)
- Albino (Idle)
- Cinnamon (Alert)
